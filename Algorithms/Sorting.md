## Bubble sort
- A na√Øve approach to sorting any comparable values in a list.
Python code snippet:
```python
def bubblesort(arr):
	for i in range(len(arr))
		for j in range(len(arr)-1)
			if arr[j] < arr[j+1]:
				temp = arr[j]
				arr[j] = arr[j + 1]
				arr[j + 1] = temp
```
